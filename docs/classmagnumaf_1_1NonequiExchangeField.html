Detailed Description

Nonequispaced Exchange Field.

The second derivative for a nonequispaced mesh can be written as [1, eq. 1.4]:

\[ f_i^{''} = \frac{2 \Big[f_{i-1} + \frac{h_i}{h_{i-1}} f_{i-1} - \big(1 + \frac{h_i}{h_{i-1}} f_i\big)\Big]}{h_i h_{i-1} \big(1 + \frac{h_i}{h_{i-1}}\big)} - \frac{h_i - h_{i-1}}{3} f_i^{'''} + ...\]

We calculate the second derivative skipping the term containing f_i as it drops out in the cross product of the LLG:

\[ f_i^{''} = \frac{2 \Big[f_{i-1} + \frac{h_i}{h_{i-1}} f_{i-1}\Big]}{h_i h_{i-1} \big(1 + \frac{h_i}{h_{i-1}}\big)}\]

[1] "A simple finite-difference grid with non-constant intervals" by Hilding Sundqvist & George Veronis
