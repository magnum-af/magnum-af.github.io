enum magnumaf::math::TruncateOutput

Truncate ghost boundary cells introduced due to boundary conditions.

This reduces the size of the output by 2 in x,y and z.

Central finite difference, computed via convolution.

Boundary condition can by handled by providing ghost cells on expaned input array with size [nx + 2, ny + 2, nz + 2, : ] and setting TruncateOutput::on, the resulting af::array has reduced dims [nx, ny, nz, : ].

Parameters
ainput array. When incorporating BC ghost cells with dim [nx + 2, ny + 2, nz + 2, : ], set TruncateOutput::on.
grid_spacinguniform grid spacing between each finite difference interval.
axisAxis enum along which to perform the finite difference, either x, y or z.
trunc_outswitch wether output dims is are reduced.

Central difference is calculated along dim via

\[ \frac{f(x_{i+1}) - f(x_{i-1})}{2h} \]
