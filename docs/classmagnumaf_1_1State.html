af::array magnumaf::State::Ms_field

Inhomogeneous, mesh dependent saturation magnetization defined at every node in units of [J/T/m^3].

Is impicitly set and used when magnetization has values of norm 0. return Ms as af::array. If Ms_field is not set, scalar Ms is tiled to dims [nx, ny, nz, 1].
